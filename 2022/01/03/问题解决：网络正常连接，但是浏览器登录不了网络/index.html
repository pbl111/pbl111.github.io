问题解决：网络正常连接，但是浏览器登录不了网络

Created2022-01-03|Updated2022-01-07
|Post View:

电脑能连网但是浏览器无网络连接

问题描述

   此时和浏览器有关的程序或者网页是无法运行的，但是非浏览器的的文件是可以运行的。

原因：

   大概率是电脑关机前没有关闭应该关闭的程序

解决方案

方案一：

  • 进入控制面板
  • 点击网络和internet
  • 点击Internet选项下面三个中的任意一个
  • 显示Internet属性 -> 点击连接 -> 点击局域网设置 -> 将“为LAN使用代理服务器（这些设置不用于拨号或VPN连接）（X）”前面的小框里的勾去掉 -> 确定即可

方案二：

  • 右键点击左下角win图标 -> 打开电脑设置 -> 打开代理 -> 关闭手动设置代理

方案三：

  • 打开上次关机前没有关闭的程序试试
