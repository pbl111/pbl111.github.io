VS code使用

打开侧边预览

   ctrl+shift+p -> 输入markdown -> 打开侧边预览 -> ok

markdown语法

  • ”#+空格+文字“ 一级标题

  • “##+空格+文字” 二级标题

  • “###+空格+文字” 三级标题

  • 段首空格的显示：“&+ensp+;+空格+文字”
